# Portfolio Projects

This directory contains example projects showcasing various web development skills.

## Projects

### 1. Color Memory Game
**Location:** `color-memory-game/`

A browser-based memory game inspired by Simon Says. Players must remember and repeat increasingly complex color sequences.

**Technologies:**
- Vanilla JavaScript (ES6+)
- HTML5
- CSS3
- Web Audio API
- LocalStorage

**Skills Demonstrated:**
- Game logic and state management
- DOM manipulation
- Event handling
- Audio programming with Web Audio API
- Object-oriented programming
- CSS animations

[Play the game](./color-memory-game/index.html) | [View README](./color-memory-game/README.md)

## Running the Projects

All projects are built with vanilla JavaScript and require no build process. Simply open any `index.html` file in a modern web browser to run the project.

## Technologies Overview

Across these projects, the following skills are demonstrated:

- **Languages:** JavaScript (ES6+), HTML5, CSS3
- **APIs:** Web Audio API, LocalStorage API, Blob API, Fetch API
- **Patterns:** Object-Oriented Programming, Async/Await, Event-Driven Architecture
- **Design:** Responsive Design, CSS Grid, Flexbox, Animations
- **Tools:** Git, Modern Browser APIs

## Adding More Projects

To add a new project:

1. Create a new directory under `projects/`
2. Include at minimum: `index.html`, CSS files, JavaScript files
3. Add a `README.md` describing the project
4. Update this main README with project details
5. Update the portfolio's `src/lib/Projects.svelte` to display the new project
